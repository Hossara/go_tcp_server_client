# GO TCP basic client and server
A practical project to learn how a TCP server and client works in GoLang

Project structure:
```text
. Project (root)
├── main.go (Project runner)
├── utils (Project utilities)
│   └── utils.go (Contains Decode and Encode Functions)
├── server
│   └── server.go
├── client
│   └── client.go
```
